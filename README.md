## 🚀 Highlighted Projects

### 🧠 [Mental Health Predictor App](https://github.com/Hardikabrol8)
> **Stack:** Python, CatBoost, Scikit-learn, Streamlit, SHAP
- Developed an **interpretable ML classification system** using CatBoost to predict clinical mental health treatment likelihood.
- Engineered feature selection utilizing Cramér's V correlation and Chi-square statistical significance tests.
- Integrated model explainability with **SHAP values** to identify and surface key prediction factors.
- Deployed a responsive Streamlit web application serving **200+ users**.

### 📊 [TOPSIS Decision Analysis Tool](https://github.com/Hardikabrol8/Topsis)
> **Stack:** Python, Flask, Matplotlib, MCDM
- Engineered a Multi-Criteria Decision Making (**MCDM**) engine using the TOPSIS algorithm to rank alternatives based on weighted performance metrics.
- Packaged the core algorithm as a **reusable Python library** and created a Flask web service for remote API access.
- Generated comparative visualizations of model rankings using Matplotlib.

### 🚲 [Bike Rental System (DBMS Project)](https://github.com/Hardikabrol8)
> **Stack:** MySQL, SQL, PL/SQL, Database Design
- Architected a normalized relational database schema (optimized to **BCNF**) for managing bike rental workflows.
- Developed automated **PL/SQL triggers** for cost calculation and concurrency control to avoid double-bookings.
- Developed stored procedures to handle rental bookings, cancellations, and availability search.
